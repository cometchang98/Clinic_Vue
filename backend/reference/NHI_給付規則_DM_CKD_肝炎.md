## A. 家醫 2.0 糖尿病申報碼（115 年起取代 P 碼）

## B. SGLT2i 糖尿病給付（Forxiga / Jardiance / Steglatro）

- **前提**：已用最大耐受劑量 **Metformin** 仍控制不佳
- **HbA1c**：單方目前無絕對門檻；115年修訂後初診 **HbA1c >7.5%** 可與 Metformin 同時給付
- **排他性**：SGLT2i / DPP4i / 兩者複方 **三擇一**（不可 SGLT2i+DPP4i 同開）
- **劑量**：每日限 1 粒
- **放寬直接給付**：重大心血管疾病；或 DKD 且 **eGFR<60 或 UACR≥30**

## C. SGLT2i/DPP4i 複方（Glyxambi / Qtern / Steglujan）— 三條件全滿足

1. 已用最大耐受劑量 Metformin
2. 已併用複方中單一成分 **≥6個月**（115年修訂放寬為 **3個月**）
3. 上述後 **HbA1c 仍 >7.5%**；每日限 1 粒

## D. GLP-1 RA 糖尿病給付（Ozempic / Trulicity / Victoza / Rybelsus）— 最嚴

- **HbA1c 仍 >8.5%** 才給付
- 前置：最大耐受 Metformin±SU，**再併用** SGLT2i/DPP4i/複方/胰島素 **其一滿6個月** 仍未達標
- **不得與 DPP4i 或 SGLT2i 合併給付**；**無 BMI 門檻**
- **豁免**：重大心血管事件者 → 免8.5%門檻、免6個月等待，直接給付

## E. SGLT2i 用於 CKD（非糖尿病適應症）

- **品項**：Forxiga、Jardiance（限 10mg）
- **eGFR 25~60** 起始；惡化至 **<15 停止給付**
- **UACR 200~5000 mg/g**
- 須加入 **Early-CKD 或 Pre-ESRD 方案**；先穩定用最大耐受 **ACEI/ARB ≥4週**
- 排除：第1型糖尿病、多囊腎、狼瘡腎、ANCA血管炎、近期化療/移植/心血管事件
- 合併 T2D → 走 DKD 規範（eGFR<60 或 UACR≥30 擇一即可，較寬）

## F. B/C 肝 115年新方案（「肝炎醫療給付改善方案」）

**行政/申報面（手冊明確收錄）**：
- 收案放寬：6個月內主或次診斷 B/C 肝、就醫≥2次；**新增 F3 以上代謝性/酒精性肝炎**
- 追蹤 **每年2次→3次**；管理費 **100→150 點**
- **VPN 系統新增必填：AFP 檢驗結果 + 肝纖維化期別**
- 品質指標：修正腹部超音波進步率、新增「失聯率」

**臨床細節（✅ 2026-06-16 已透過 NotebookLM 外網研究補入 70 個來源，附引用）**：

**FIB-4 指數**（公式：`年齡 × AST ÷ [血小板(10⁹/L) × √ALT]`）
| 分級 | FIB-4 |
|---|---|
| F0 無纖維化 | < 1.3 |
| F1 輕度 | 1.3 ~ 2.0 |
| F2 中度 | 2.1 ~ 3.24 |
| F3 重度 | 3.25 ~ 6.4 |
| F4 肝硬化 | ≥ 6.5 |
- 排除 advanced fibrosis：<1.45（NPV 90%）；預測：>3.25（特異度 97%）

**APRI 指數**（公式：`(AST ÷ AST上限 × 100) ÷ 血小板(10⁹/L)`）
- 顯著纖維化（≥F2）：排除 <0.5、確診 >1.5（部分研究 >0.7）
- 肝硬化（F4）：排除 <1.0、確診 >2.0（特異度 ~90%）

**AFP（肝癌篩檢）**
- 正常 <20；灰區 20~400（需搭超音波）；高度警惕 >400（或 >400持續4週 / >200持續8週）
- 依病因調整：HCV+ 切點 59、HCV− 切點 11
- 頻率：高危群（B/C肝、肝硬化）每 6 個月；肝癌家族史 45歲起每6月；一般50歲以上每年

**超音波 + AFP 監測間隔**
- 常規（AASLD/EASL/APASL/TLCA）：慢性 B/C 肝、各原因肝硬化 → **每 6 個月**腹部超音波 ±AFP
- 極高風險（JSH：B肝或C肝併肝硬化）→ **每 3~4 個月**；台灣建議嚴重纖維化/已確診HCC的 AFP/PIVKA-II 至少每3月（年4次）

## 防呆邏輯對照（給 M3 區塊4 實作用）

| 病患情境 | 系統應提示 |
|---|---|
| 開 SGLT2i 但無 Metformin 病史 | ⚠️ 健保要求先用最大耐受 Metformin |
| 同時開 SGLT2i + DPP4i | 🚫 三擇一，不可同開 |
| 開 GLP-1 RA 但 HbA1c <8.5% 且非心血管豁免 | ⚠️ 未達 8.5% 門檻，恐核刪 |
| 開 SGLT2i 治 CKD 但無 ACEI/ARB ≥4週 | ⚠️ 需先穩定 RAS 阻斷劑 |
| DM 追蹤當次未開 DM 藥 | 🚫 家醫2.0 追蹤加成費不給付 |
